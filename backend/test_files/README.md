# MPC 功能测试文件

本目录包含用于测试 MPC 私密验证功能的模拟文件。

---

## 📁 文件列表

| 文件名 | 说明 |
|--------|------|
| `公司A_银行流水_测试.pdf` | 模拟公司A的银行流水 |
| `公司A_承诺书_测试.pdf` | 模拟公司A的承诺书 |
| `公司B_银行流水_测试.pdf` | 模拟公司B的银行流水 |
| `公司B_承诺书_测试.pdf` | 模拟公司B的承诺书 |

---

## 🎯 测试场景

| 公司 | 模拟流水金额 | 信用档次 |
|------|--------------|----------|
| 公司A | 5000万 | 发展级 (1000万-1亿) |
| 公司B | 1.5亿 | 成熟级 (1亿-10亿) |

**预期结果**：公司B 的流水档次更高

---

## 🔧 使用方法

### 测试步骤

1. **打开两个浏览器窗口**
   - 窗口1：模拟公司A
   - 窗口2：模拟公司B（可使用隐私窗口）

2. **公司A（窗口1）**
   - 访问 http://localhost:8000
   - 选择"创建新会话"
   - 输入公司名称：`测试公司A`
   - 选择隐私级别
   - 复制会话 ID

3. **公司B（窗口2）**
   - 访问 http://localhost:8000
   - 选择"加入会话"
   - 粘贴会话 ID
   - 输入公司名称：`测试公司B`

4. **两边分别上传文件**
   - 公司A上传：`公司A_银行流水_测试.pdf` + `公司A_承诺书_测试.pdf`
   - 公司B上传：`公司B_银行流水_测试.pdf` + `公司B_承诺书_测试.pdf`

5. **提交承诺**
   - 公司A填写流水金额：`50000000`（5000万）
   - 公司B填写流水金额：`150000000`（1.5亿）
   - 两边分别提交

6. **查看结果**
   - 两边点击"查看比较结果"
   - 验证结果是否显示公司B档次更高

---

## 💡 注意事项

- 测试文件为模拟数据，仅用于功能测试
- 系统只检查文件扩展名，不验证文件内容
- 可使用任意 PDF/PNG/JPG 文件替代测试
- 关键是填写的流水金额，这决定了档次计算
