# 银行流水文件说明

---

## 什么是银行流水

银行流水是指企业银行账户的交易记录，包含一定时期内的所有收入和支出明细。银行流水是评估企业经营状况和信用等级的重要依据。

---

## 文件要求

### 基本要求

| 项目 | 要求 |
|------|------|
| 时间范围 | 最近 12 个月（完整年度） |
| 账户类型 | 企业基本户或一般户 |
| 文件格式 | PDF、JPG、PNG |
| 文件大小 | 不超过 10MB |
| 清晰度 | 文字清晰可辨 |

### 必要信息

银行流水文件应包含以下信息：

- ✅ 账户名称（与企业名称一致）
- ✅ 银行账号
- ✅ 开户银行
- ✅ 交易日期
- ✅ 交易摘要
- ✅ 交易金额（收入/支出）
- ✅ 账户余额
- ✅ 银行盖章（原件或电子章）

---

## 获取方式

### 方式一：银行柜台打印（推荐）

1. 携带企业公章、营业执照、法人身份证
2. 前往开户银行柜台
3. 申请打印对公账户流水
4. 银行盖章确认
5. 扫描成 PDF 文件

**优点**：具有法律效力，银行盖章确认

### 方式二：网上银行导出

1. 登录企业网上银行
2. 进入账户管理 → 交易明细查询
3. 选择时间范围（最近12个月）
4. 导出 PDF 或 Excel 格式
5. 如需盖章，到银行柜台补章

**优点**：方便快捷，随时可导出

### 方式三：手机银行导出

1. 登录企业手机银行 APP
2. 查询交易明细
3. 选择时间范围
4. 导出或截图
5. 整理成 PDF 文件

**优点**：随时随地可操作

## 信用档次对照表

根据年度银行流水总额，系统自动计算企业信用档次：

| 档次 | 名称 | 年度流水范围 | 典型企业规模 |
|:----:|------|--------------|--------------|
| 1️⃣ | 初创级 | 10万 - 100万 | 初创企业、小微企业 |
| 2️⃣ | 成长级 | 100万 - 1000万 | 成长期企业 |
| 3️⃣ | 发展级 | 1000万 - 1亿 | 发展期企业、中型企业 |
| 4️⃣ | 成熟级 | 1亿 - 10亿 | 成熟期企业、大型企业 |
| 5️⃣ | 领军级 | 10亿以上 | 行业领军企业、上市公司 |

## 计算说明

### 流水金额计算方法

**年度流水 = 年度总收入（贷方发生额）**

> ⚠️ 注意：不包含以下情况
> - 账户间内部转账
> - 银行利息收入
> - 退款等冲销交易

### 示例

```
某公司 2024 年银行流水：

月份    收入金额
1月     500万
2月     480万
3月     520万
...
12月    550万
────────────
合计    6000万 → 发展级（1000万-1亿）
```

## 隐私保护

### 数据安全承诺

🔐 **您的银行流水在系统中受到严格保护：**

1. **金额不上传**：具体金额只在您的浏览器中计算，服务器只知道档次等级
2. **加密传输**：所有文件通过 HTTPS 加密传输
3. **即时删除**：比较完成后文件自动删除
4. **不存储明细**：服务器不存储任何交易明细

### 脱敏建议

如果您希望进一步保护隐私，可以在上传前对流水进行脱敏处理：

- 隐藏部分交易摘要
- 模糊处理对方账户信息
- 保留关键的金额和日期信息

## 常见问题

### Q1: 必须上传真实的银行流水吗？

MPC 模块目前不验证银行流水的真实性，系统仅检查文件格式。但您需要签署《流水真实性承诺书》，虚假承诺将承担法律责任。

### Q2: 银行流水会被对方看到吗？

不会。系统采用密码学承诺方案，对方只能知道比较结果（谁的档次更高），无法获知您的具体流水金额或详细交易记录。

### Q3: 多个银行账户怎么处理？

您可以：
1. 分别打印各银行流水
2. 合并成一个 PDF 文件
3. 自行计算总流水金额
4. 在提交时填写合计金额

### Q4: 流水不足一年怎么办？

如果企业成立不足一年，可以：
1. 提供成立至今的全部流水
2. 按实际月份计算年化流水
3. 在承诺书中说明情况
